# Task Manager Frontend (React + Tailwind)

A modern, responsive frontend for the Task Manager application.

## Features

- **Clean UI**: Minimalist design with a focus on productivity.
- **Task Management**: Create, toggle, and delete tasks.
- **Filtering**: Filter by 'Pending' or 'Completed' status.

## Setup Instructions

1. **Install Dependencies**:

   ```bash
   npm install
   ```

2. **Environment Configuration**:
   Create a `.env` file:

   ```
   VITE_API_URL=http://localhost:5001
   ```

3. **Run Development Server**:

   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```
